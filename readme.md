# BPM2MS

Convertisseur de bpm vers ms qui affiche une liste de ms en fonction de la division du tempo saisi.
Ex:

```c
    ./bpm2ms 120 
    TEMPO: 120
    DIV |   MS
    1/1 |   2000
    1/2 |   1000
    1/4 |   500
    1/8 |   250
    1/16|   125 
``` 

Pour lancer un calcul, il faut entrer la commande: `./bpm2ms tempo`
